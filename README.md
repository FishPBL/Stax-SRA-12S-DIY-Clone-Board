Electrostatic amps are extremely dangerous and have VERY HIGH voltages inside them. I am not responcible for your safety if you choose to build this. 
Eveything is provided with the Apache 2.0 Liscence, which means you can do pretty much whatever you want with this. 

This is all the stuff you need in order to make a clone of a Stax SRA-12S. 
Currently the only board I have replicated so far is the main amplification board under Stax_SRA-12S_PB-300_Main_Amp_PCB_Fabrication_File.zip.
Just download that file, and go to a PCB makers website (Makers include JLCbcp, PCBway, DKRed(Digikey), and OSHPark) and upload the entire file.
Do not extract the file, just dump the whole thing on there. 

All of the other files are suplimentary, and include pictures of real PCBs from the wild, schematics from Stax (That usually have some mistakes on them) and a service manual that has a pretty decent BOM on it.
I'll be updating to add more PCBs and maybe an updated schematic at some point.
